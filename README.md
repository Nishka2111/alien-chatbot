# AlienBot Chatbot 🤖👽

A simple rule-based chatbot built in Python that simulates a conversation with an alien exploring Earth.

## Features
- Greets the user and asks their name
- Uses regex to detect user intent
- Responds to questions about the alien's planet
- Explains why the alien is visiting Earth
- Can cube numbers provided by the user
- Handles unknown inputs with generic responses

## Technologies Used
- Python
- Regular Expressions (re)
- Random module

## Example Interaction

Hello there, what's your name?  
Nishka  

Hi Nishka, I'm Etcetera. I'm not from this planet.  
Will you help me learn about your planet?  

yes  

Why are you here?  
why are you here  

I come in peace.
